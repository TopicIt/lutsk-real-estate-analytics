# PROJECT_CONTEXT.md

I am building a personal website/portfolio for Dmitro, based in Lutsk, Ukraine.

The site is a simple personal landing page with sections like:

* About me
* Services
* Projects
* Contacts

The current style is minimal, clean, light background, black text, simple cards, rounded buttons, and Ukrainian language content.

I want to add a separate useful project/page to this website:

**Real Estate Market Analytics for Lutsk**

The main idea is to collect real estate listing data from public sources and display market trends as charts.

The first version should be simple and working, not perfect.

Primary goal:
Create an MVP that tracks how the number of real estate listings changes over time in Lutsk.

Later goals:

* average rental price;
* average sale price;
* average price per square meter;
* category comparison;
* investment/rental yield calculator;
* possible price prediction in the future.

The project should be easy to extend.
